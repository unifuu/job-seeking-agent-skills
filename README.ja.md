# Job Seeking Skills (Claude Code)

[English](README.md)

このリポジトリは、**Claude Code** のスキルを活用して、就職活動における企業調査プロセスを効率化・自動化するために設計されています。Obsidian と互換性のある構造化された Markdown レポートを生成します。

## 🚀 特徴

- **自動企業調査**: 企業の基本情報、技術スタック、採用要件などを深く掘り下げます。
- **ステータス管理**: 応募状況や履歴を一元管理します。
- **スマートランキング**: 希望条件に基づいて企業をスコアリング・ランク付けします。
- **求人分析**: 求人URLを分析し、包括的な面接戦略を生成します。
- **進捗レポート**: 活動状況を可視化し、次のアクションを明確にします。
- **Obsidian 対応**: フォーマットされたレポートを直接出力します。

## 🛠 利用可能なスキル

| スキル                      | 説明                                    | トリガー (例)                                |
| --------------------------- | --------------------------------------- | -------------------------------------------- |
| **企業調査** (Research)     | 企業データと社員インタビューを収集      | `株式会社[企業名]について調査`               |
| **ステータス管理** (Status) | 応募状況を更新 (応募済み、面接予定など) | `[企業名] 一次面接予定`, `[企業名] 書類通過` |
| **求人分析** (JD Analysis)  | 求人URLを分析し面接戦略を作成           | 求人URLを直接入力                            |
| **進捗レポート** (Report)   | 活動状況の概要・統計・ランキングを生成  | `レポート作成`, `進捗レポート`               |

## 📁 構成

- `companies/`: 生成された企業調査レポート (`[企業名].md`)
- `strategies/`: 求人分析と面接戦略
- `reports/`: ステータス・進捗レポート
- `about_me/`: あなたのプロフィールと希望条件
- `.claude/skills/`: カスタムスキル定義

## 💡 使用例

### 1. 企業を調査する

```bash
株式会社[企業名]について調査して
```

_`companies/株式会社[企業名].md` を生成_

### 2. ステータスを更新する

```bash
株式会社[企業名] 書類選考通過
```

_企業ファイルのステータスを更新_

### 3. 求人を分析する

```bash
https://www.green-japan.com/company/xxxx/job/xxxxx
```

_`strategies/[企業名].md` に包括的な面接戦略を生成_

### 4. 進捗を確認する

```bash
進捗レポートを作成
```

_活動全体のサマリーレポートを作成_
